# 說明
* 非資通系統, 在個人電腦即可執行, 只是小工具, 不用系統分級, 掃瞄時只要在同網段即可
* 只要下載 ipmac.exe 執行檔即可執行, 在本機 http://localhost:9999 即可使用
* step 1: 先在"系統設定"功能裡,設定要掃瞄的網段CIDR, PS:掃瞄時要同網段(因為使用arp scan)
* step 2: 使用"網路掃瞄"功能,完成後,在"掃瞄記錄"中, 可以將結果匯入到"IP/MAC維護"表裡
* step 3: 在 "IP/MAC維護" 中可以對每筆記錄增加註解, 如 2F茶水間印表機, 以方便記憶這個IP是做什麼用途
* step 4: 下次進行"網路掃瞄"時, 若發現IP和MAC有不同時, 會顯示警告訊息
* 支援發現MAC address變動時,自動寄信功能

PS:要用管理員身份帳號執行
